# React Portfolio Website

---

This is my updated personal website where I showcase things about me and my work created using React + Typescript. This website uses Tailwind for the responsive styling, and Framer Motion for the animation.

## Installation

Install my website with npm

```bash
  git clone https://github.com/fedexid/personal-web
  npm install
```

## Tech Stack

- Vite + React
- Typescript
- Tailwind CSS
- Framer Motion
- Netlify
